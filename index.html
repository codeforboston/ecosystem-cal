Boston's civic innovation ecosystem is growing rapidly! Post your civic tech event on this calendar, and share it with the rapidly growing number of organizations in this space.

When you submit an event, Code for Boston will review it promptly and post it (or contact you if there are broken links, etc.).

About the calendar: The need for a civic innovation calendar was highlighted when four civic tech events were all scheduled for the same weekend. We want to enable all the civic tech groups in the Metro Boston area to share their events with each other, forming a communicative network where overlaps happen rarely and attendance is assured! Code for Boston built and moderates the calendar. We are committed to process transparency and will always respond to questions.
